Privacy

{% translate "Review how Schedama.com handles data" %}

{% translate "User privacy is important on Schedama!" %}

{% translate "That's why we do not ask for or collect any personally identifiable information (eg.: names, email addresses, IP addresses, etc.)." %}

{% translate "Types of data collected" %}

{% translate "Schedama only collects data related to events created by users and only data entered by users." %}
{% translate "Such data may include:" %}

  • {% translate "Event title and description (if present)" %}
  • {% translate "Location (if provided)" %}
  • {% translate "Date, time and duration of the event" %}
  • {% translate "List of participants" %}
  • {% translate "Voting preferences submitted by participants" %}
{% translate "To better protect your privacy, we recommend that you do NOT include personal or confidential information in the events you create." %}

{% translate "How we use data" %}

{% translate "Data related to events created are saved in our database with the sole purpose of:" %}

  • {% translate "Enable event authors to view, manage and share information" %}
  • {% translate "Allow participants to view relevant information about the event and indicate their participation (possibly by voting for one or more of the available dates)" %}

{% translate "Data sharing" %}

{% translate "Upon event creation, two links are generated:" %}

  • {% translate "Participant URL: Anyone with the link can view the event (event authors can share this link with the participants)" %}
  • {% translate "Administration URL: Anyone with the link can modify the event (we recommend event authors to keep this link to themselves or share it only with trusted individuals)" %}

{% translate "The links are unique and are generated based on the event ID." %} {% translate "These different links represent the two ways an event can be shared." %}
{% translate "It is the user's responsibility to keep these links and share them only with the intended recipients." %}

{% translate "History" %}

{% translate "When you create or visit an event, its links are made available on the history page." %}
{% translate "This is particularly useful in case you can no longer find the link to an event or if you've forgotten to save it." %}
{% translate "History data only reside locally on your device in your browser storage and are NOT sent or saved by Schedama or shared with third parties." %} {% translate "History data are therefore not permanent and may be lost if the browser data are deleted." %}

{% translate "Data retention" %}

{% translate "Event data are retained until the author of the event deletes them." %}
{% translate "To delete your event data:" %}

  • {% translate "Visit the administration page of your event" %}
  • {% translate "Scroll to the bottom until the 'Danger Zone' area" %}
  • {% translate "Read the disclaimer and select 'I understand'" %}
  • {% translate "Click on 'Cancel event'" %}
  • {% translate "After approximately 2 days the event and all it's data will be automatically and permanently deleted" %}
